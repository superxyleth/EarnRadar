# EarnRadar 活动雷达

EarnRadar 是一个静态网页，用于集中查看 Binance / OKX 的理财、Alpha、闪赚、Boost、链上赚币和交易奖励活动。

公开访问链接：

https://superxyleth.github.io/EarnRadar/

## 功能

- 按交易所查看 Binance / OKX 活动
- 按模块筛选理财、Alpha、闪赚、Boost、链上赚币和交易任务
- 展示活动时间、奖励规模、参与门槛、规则摘要和官方公告入口
- 根据活动结束时间自动隐藏已过期活动
- 支持关键词搜索

## 项目结构

- `index.html`：页面结构和交互逻辑
- `styles.css`：页面样式和响应式布局
- `data.js`：活动数据源
- `assets/`：Logo 和教程素材

## 说明

活动信息来自 Binance / OKX 官方公告或官方活动页面。页面仅用于活动聚合和信息整理，参与前请以对应官方页面的最终规则、地区限制和风险提示为准。
